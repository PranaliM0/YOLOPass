# YOLOPass 🎫 - Event Registration Platform

YOLOPass is a full-stack event management and registration platform that supports three roles: **Admin**, **Organizer**, and **Attendee**.

---

## 🔧 Tech Stack

- **Frontend:** React.js
- **Backend:** Ruby on Rails (API mode)
- **Database:** PostgreSQL

---

## 👥 Roles & Features

### 🛠 Admin
- Full platform access
- Manage all users and events

### 🎤 Organizer
- Create and manage events
- Apply discounts, early bird pricing
- Monitor registrations

### 👤 Attendee
- Register for events
- Add multiple participants per registration
- Upload ID proofs
- Use discount codes or early bird offers
